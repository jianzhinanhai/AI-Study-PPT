# Travel Itineraries

用于保存和发布旅行行程 HTML 文件。

## 目录结构

- `index.html`：行程库首页
- `trips/<trip-slug>/index.html`：每个独立行程
- `.github/workflows/deploy-pages.yml`：GitHub Pages 自动发布

## 发布
