Operating process

三、端到端执行流程：从伙伴准备到审核、付款与持续绩效

标准化操作的重点是前置锁定基线、尺码、证据和时限，而不是在交付结束后补材料。

PHASE 01

伙伴级准备

  • MCI enrollment、协议、税务、银行
  • 确认 Direct / Indirect / LSP / FDD
  • 核验 designation、specialization、TTM
  • 付款义务履行率 ≥95%
持续维护
PHASE 02

机会筛选

  • 客户 territory 与客群资格
  • 产品、订阅期、净新增与 COCP
  • 选择正确 offer 与 t-shirt size
  • 锁定 ACR / MAU / MPU / seat 基线
Nomination 前
PHASE 03

Claim 与 Consent

  • 建立 claim 并请求客户同意
  • 记录 Partner of Record
  • 确认 program cap 与 active claim capacity
  • 形成证据清单和责任人
客户通常 30 天响应
PHASE 04

交付与绩效达成

  • 按 engagement 范围完成交付
  • 满足 seat / MPU / ACR / workload mix
  • 不得拆单、复制或跨客户复用证据
  • 持续记录 before / after
90 / 120 / 200 / 260 天
PHASE 05

POE 与审核

  • Customer attestation
  • Partner survey
  • Invoice / CSR
  • 专项 dashboard 与交易证据
Conversion 需等待 ≥30 天
PHASE 06

付款与治理

  • 处理 Action Required / Dispute
  • 保存 POE 至少 24 个月
  • 监控 success rate 与 claim ratio
  • 处理 clawback / hold / forfeiture
按项目与 earning period

Restricted fund mechanics

五、Co-op 机制：先赚取、后发生费用、再据实报销

Co-op 的本质是有期限、有用途、有证据要求的受限资金，不等于当期现金返点。

1. Earn从 CSP / Hosting earning 中形成
2. Threshold达到 USD 10,000 后进入 claims tool
3. Spend下一半年度发生合格活动费用
4. Claim & POE活动后 45 天内申领并提交证据
5. Reimburse按四项取低，未用余额作废
Reimbursement = min(Available Co-op, Eligible Actual Cost, Category Cap, Market Rate Cap)
50%Allocated Resources（非 Surface）
5%Partner AI Adoption / Customer Adoption
3%Marketing Automation
1–2%Merchandise / SPIFF / Internal Incentive
时间机制：2026H1 赚取 → 2026H2 使用，Claim 截止 2027-02-15；2026H2 赚取 → 2027H1 使用，Claim 截止 2027-08-15。未使用资金不 rollover、不 reinstatement。

可报销方向

  • 伙伴培训、认证与第三方审计
  • Partner AI Adoption / Customer Zero
  • 活动、Paid Media Multi-Touch、Targeted Prospecting
  • Allocated Sales / Marketing / Technical Resources
  • Customer Adoption、Workshop、Co-Innovation

不可报销方向

  • 一般运营、overhead、基础设施、折旧资产
  • 重复申领、跨项目申领、未授权差旅
  • Microsoft license / support、折扣与 credit note
  • 竞争产品、BOM / cost of goods、娱乐费用
  • 超过 fair market value 的费用
Calculation engine

六、计算模型：把“收入、受限资金、成本与风险”分开

预算时应同时计算确定收入、概率加权 pipeline、交付成本、不可报销费用和 clawback 风险。

Approved Activity Fees+ Approved Conversion Bonuses+ CSP Rebate Cash+ Advisor / Enterprise Fees+ Claimed Co-op Reimbursement− Clawbacks / Forfeitures

固定活动费

Activity Fee = Eligibility Flag × POE Approval Flag × Fixed Rate

概率加权收入

Expected Income = Rate × P(Eligibility) × P(Consent) × P(Delivery) × P(POE)

Biz Apps Advisor

max(Current Net Paid Units − Prior HWM, 0) × Rate Card

CSP

Core Base×Rate + Strategic Base×Rate + Eligible YoY Growth×Rate

Enterprise

Eligible Revenue×Lever + Public Sector Revenue×Accelerator + min(ACR×1.5%, $200K)

净贡献

Expected Incentive − Delivery Cost − Non-reimbursable Expense − Expected Clawback
